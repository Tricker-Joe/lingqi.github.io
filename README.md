# GitHub Pages Personal Academic Website

This is a static personal academic homepage designed for GitHub Pages.

## Files

- `index.html`: main single-page site
- `styles.css`: visual styling and responsive layout
- `assets/miniature-robot-lab.png`: hero image

## Replace Before Publishing

- Google Scholar link
- YouTube link
- bilibili link
- Professor profile links

Search for `href="#"` in `index.html` to replace placeholder URLs.

## Publish With GitHub Pages

1. Create a GitHub repository, for example `lingqitang.github.io`.
2. Upload the files in this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for deployment.
7. Your site will be available at `https://<your-github-username>.github.io/`.

If you use a repository name other than `<username>.github.io`, the site URL will be:

`https://<your-github-username>.github.io/<repository-name>/`
